# Aamchi Mumbai 🗺️❤️

> **Explore. Guess. Challenge. Discover Mumbai.**

Aamchi Mumbai is a social location-guessing game built around the city of Mumbai. Inspired by exploration games like GeoGuessr, Aamchi Mumbai combines real-world discovery, local knowledge, community-generated challenges, and friendly competition into a single platform.

The game transforms Mumbai into a giant interactive playground where players can test their knowledge of the city's streets, landmarks, neighborhoods, culture, architecture, hidden gems, and everyday life.

Unlike traditional location-guessing games that rely solely on Street View imagery, Aamchi Mumbai introduces a community-driven challenge system where players create and share their own location challenges using real photographs taken across the city.

# Game Modes

## Street View Mode

The classic exploration mode.

Players are dropped into a random Mumbai location using Street View.

### Gameplay

* Move around the area.
* Observe surroundings.
* Collect clues.
* Place a pin on the map.
* Receive a score based on accuracy.

### Skills Tested

* Observation
* Geography
* Pattern recognition
* Local knowledge

### Features

* Solo play
* Daily challenges
* Ranked gameplay
* Timed rounds
* Accuracy tracking

---

## Community Challenge Mode

The flagship feature of Aamchi Mumbai.

Players upload photographs taken across Mumbai and challenge others to identify the location.

### Why It Exists

Street View only covers locations available through Google.

Community Challenges allow players to showcase:

* Hidden spots
* Local favorites
* Unique viewpoints
* Temporary installations
* Rare locations

This creates an ever-growing library of challenges generated by the community.

---

# Community Challenge Workflow

## Challenge Creation

Player uploads an image.

System:

* Reads GPS coordinates from image metadata
* Stores coordinates securely
* Removes metadata from public image
* Applies privacy protections
* Generates challenge

Players never see the embedded coordinates.

---

## Challenge Solving

Other players:

* View image
* Analyze clues
* Place guess on map
* Receive score

The closer the guess, the higher the score.

# Scoring System

Scoring depends on:

* Distance from actual location
* Time taken
* Challenge difficulty

### Factors

#### Accuracy

Closer guesses earn higher points.

#### Speed

Faster answers receive bonuses.

#### Difficulty

Harder challenges reward more points.

High-quality challenges:

# Privacy & Safety

User privacy is important, especially for community-generated content.

---

## Image Processing

Before publication:

* Faces are blurred
* Vehicle number plates are blurred
* Metadata is removed from public images

---

## Metadata Handling

GPS information is:

* Extracted during upload
* Stored securely
* Never exposed to players
